# new-horizon
Simple tabletop roleplaying system

## Goal
Intuitive character building tool where a character exists on the basis of cards.

## Plan

* Searchable cards
  * Live update while typing
* Print selection
* At least 3 styling options
* Admin login
  * CRUD functions
    * Add new cards
    * Update cards
    * Delete cards
    * Deactivate cards
* History of changes
